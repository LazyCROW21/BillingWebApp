
# BillingWebApp
Web App for Khodiyar Diary Parlor Patan

## App Feature

 - Customer Page to shop
- Admin Panel
-- Create Bill
-- View Bill/ E-Mail bill/ Bill as PDF
-- Add Item
--  Edit/remove items
-- Manage Customers
-- Export data as excel sheet

## Technology Stack: 
- Php + Phpmailer + fpdf.
- HTML/CSS/JS
